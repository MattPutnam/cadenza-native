# Cadenza

Cadenza is a live MIDI performance manager for programming the complicated keyboard parts of modern musicals. The most direct competitor is Apple MainStage. It differs from MainStage in the following ways:

## Hardware synthesis

If you need the power of software synthesis, then you should be using MainStage. If you just need to dial up some sounds and go, MainStage is overkill. Cadenza works with the sound banks on your digital piano or an external sound module such as the Roland JV-1080. Many shows from the '80s through '00s were orchestrated for these modules, so you end up with exactly what the orchestrator intended. Hardware synths are also fast and reliable--no random slowdowns or crashes due to memory issues.

## Built for mobile

Now that we're not doing software synthesis, there's no reason to use an expensive laptop. Cadenza runs on iOS and Android, simplifying your setup.

## Optimized for Musical Theatre

MainStage is marketed as a solution for any kind of live performance, and so it's full of features that musical theatre keyboardists never need, while also obscuring (or leaving to plugins) features that musical theatre keyboardists need all the time.

* Easy setup of a "next cue" pedal. Cadenza utilizes a concept called "triggers" where a wide varietey of actions can be taken on any kind of input. By default, Cadenza adds triggers that go to the next patch on right arrow, down arrow, and page down, and to the previous patch on left arrow, up arrow, and page up, consistent with most brands of bluetooth page turner pedal. It also provides quick buttons to setup using the highest/lowest keys on your keyboard, or any pedal that you plug into whatever pedal jacks your keyboard has. You can also easily set up "next cue" triggers per cue in cases where things get hectic.
* Remap control signals globally, per song, per cue, or per patch. Have a pedal that you use as an expression pedal for most of the show, but in one spot you need it to act as a volume pedal? Simply map expression to volume for that cue. Want the expression pedal to only go to one of the several patches you have loaded? Just map expression to nothing for those patches.
* Harp glissandos. Easily set notes in all octaves to transpose by any arbitrary amount (even beyond what a real harp can do).
* Ghost notes. Seen in Pippin, Spelling Bee, and many more, have one note trigger any amount of other notes when it's played.

## Integrated with the sheet music

Have your part on a PDF? Import it into Cadenza, and you can view your music and programming on one screen. Turn pages and advance cues with a single pedal.
