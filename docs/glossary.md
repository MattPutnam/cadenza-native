# Glossary

A glossary of technical terms used by Cadenza

| Term | Definition |
| ---- | ---------- |
| Keyboard | Any kind of MIDI input device, almost always a digital piano or MIDI controller |
| Synthesizer | Any kind of MIDI synthesizer, usually either a digital piano's sound bank or a synth module such as the Roland JV-1080 |
| Bank | A collection of sounds on a synthesizer. Synthesizers usually have a number of "preset" banks along with a "user" bank for user-created sounds, and possibly a separate "GM" bank for General MIDI |
| Channel | A MIDI channel, internally represented as 0-15 but displayed as 1-16 |
| Patch | The most basic sound type supported by Cadenza. Addressable on a synthesizer with a bank name + patch number. |
| Trigger | An input (or sequence of inputs) that causes Cadenza to take some action, such as advancing to the next cue |
| Cue | A preset for a particular section of a show. Comprised of a set of patches and their assignments to regions of keyboards, plus any triggers, controller overrides, etc. Sometimes called a "patch" in other products. |
